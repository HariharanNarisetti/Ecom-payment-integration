# Full-Stack E-Commerce Platform

A complete, fully functional, production-ready E-Commerce Web Application built with the MERN stack (MongoDB, Express, React, Node.js) featuring modern UI/UX design and payment gateway integrations.

## Features
- **Frontend**: React + Vite + Tailwind CSS with a stunning Glassmorphism/Neumorphism design.
- **Backend**: Node.js + Express.js API.
- **Database**: MongoDB + Mongoose.
- **Authentication**: JWT & Google OAuth.
- **Payment**: Stripe & PayPal integration.
- **Admin Dashboard**: Full management panel for products, orders, users, and analytics.

## Getting Started

### Prerequisites
- Node.js (v18+)
- MongoDB Atlas or Local instance
- Stripe, PayPal, and Cloudinary API credentials.

### Installation

1. Clone the repository
2. Install dependencies for both `server` and `client`
```bash
cd server
npm install
cd ../client
npm install
```

3. Configure Environment Variables (see `.env.example` in both client and server folders)

4. Run the development servers
```bash
# In the server directory
npm run dev

# In the client directory
npm run dev
```
